# AI Collective Hampton Roads

Community website for **AI Collective Hampton Roads** — a regional group focused on AI education, workforce development, and bringing AI practitioners together in the Hampton Roads area.

**Live site:** [aicollectivehr.org](https://aicollectivehr.org)

## Tech Stack

- **Framework:** Astro 4
- **Language:** TypeScript
- **Hosting:** GitHub Pages

## Pages

- **Home** — mission statement, upcoming events, community overview
- **About** — organization background and team
- **Events** — past and upcoming meetups, workshops, and talks
- **Connect** — join the community
- **Partnerships** — collaboration opportunities
- **Volunteer** — ways to get involved
- **Workshops** — hands-on AI learning sessions

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

## Deployment

Automatic via **GitHub Actions** on push to `main`. The workflow at `.github/workflows/deploy.yml` builds the Astro site and deploys to GitHub Pages with the custom domain `aicollectivehr.org`.
